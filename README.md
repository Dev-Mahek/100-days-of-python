# 🐍 100 Days of Python

Welcome to my **100 Days of Python** journey! 🚀

This repository documents my progress as I learn Python from the ground up. Every "Day" represents a learning milestone where I study new concepts, solve coding challenges, take notes, and build small projects.

My goal is not just to learn Python syntax but to become confident in writing clean, readable, and practical code.
